Reporting


Reporting in RegSmart is easy. You have various choices for creating reports, like picking the kind of analysis you want and deciding which databases to exclude. Exclusion databases work as filters, leaving out less important information, such as common operating system services that are always present. These databases are only used for reporting, and their main job is to make the generated report shorter by leaving out less relevant details.

Here are the option to include in the Report of the Registry analysis

  • System Analysis
  • OS Analysis
  • Network Analysis
  • Device Analysis
